### Install & Run

1. Clone this repository to your local
2. Build dockerfile
3. Run docker container

```console
$ git clone git@github.com:5zzang/5zzang.github.io.git
$ cd 5zzang.github.io
$ ./build.sh
$ ./run.sh
```
